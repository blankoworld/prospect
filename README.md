# prospect

Projet de stage pour le prospect client dans la Société Open

# Mémoire de stage

## Dépendances

  * texlive-core
  * texlive-qrcode

## Génération du rapport

Sous GNU/Linux : 

1. Installer les dépendances texlive-core et texlive-qrcode
2. Compiler de la manière suivante : 

```bash
cd rapport
make clean && make pdf
```
